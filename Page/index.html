Do you want Express to automatically load your route files recursively?

This is the simplest and fastest solution...

Automatic route loading system for Express.js applications that automatically discovers and loads routes from your routes directory with built-in analytics and performance tracking.

Get Started View Examples
app.js 
import express from 'express';
import { initializeAutoRouter } from '@ljgazzano/express-auto-router';

const app = express();

// Initialize with default routes directory
const router = await initializeAutoRouter();
app.use('/', router);

app.listen(3000);

Usage Examples

Basic Implementation 
import express from 'express';
import { initializeAutoRouter } from '@ljgazzano/express-auto-router';

const app = express();

// Initialize with default routes directory (./routes)
const router = await initializeAutoRouter();
app.use('/', router);

app.listen(3000, () => {
  console.log('Server running on port 3000');
});
Custom Routes Directory 
import express from 'express';
import { initializeAutoRouter } from '@ljgazzano/express-auto-router';
import path from 'path';

const app = express();

// Initialize with custom routes directory
const router = await initializeAutoRouter({
  modulesPath: path.join(process.cwd(), 'api', 'routes')
});

app.use('/api', router);
With Analytics & Metrics 
const router = await initializeAutoRouter({
  enableMetrics: true,
  metricsLogPath: './logs/api-metrics.log'
});

// Access metrics programmatically
const currentMetrics = router.getMetrics();
console.log('Total requests:', currentMetrics.summary.totalRequests);

// Generate detailed report
const report = router.generateReport();
